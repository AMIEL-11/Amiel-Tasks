# 📝 To-Do List App

A clean and simple to-do list web app that helps users organize tasks by priority and category.

## 🚀 Features
- Add tasks
- Set priority (High / Medium)
- Categorize tasks (Work / Personal)
- Mark tasks as completed
- Clean and minimal user interface

## 🛠️ Built With
- HTML
- CSS
- JavaScript
